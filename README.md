1、架构简介：
环境准备： 配置树莓派 5 操作系统，安装 Ollama，配置阿里云 ASR API 及外接 TTS 模块。编写可切换散热策略（风扇开/关）与触发模式（GPIO 监听/脚本直连）的控制脚本。
模型部署： 部署 Llama-3.2-1B (Q8_0) 与 Qwen3-1.7B (Q4_K_M)，实施“显存对齐”的非对称量化策略。
分层实验执行：
场景一（热抑制）： 开启风扇 -> 监控温度稳定 -> 注入 Prompt -> 记录文本质量（盲测评分）。
场景二（热演化）： 关闭风扇 -> 冷启动 -> 连续负载 -> 高频记录 (Temp, Freq, TPS) -> 绘制衰减曲线。
场景三（全链路）： 关闭风扇 -> 监听 GPIO 按键 (T_0) -> 录音/ASR/LLM/TTS -> 记录播放结束 (T_{end}) -> 计算 L_{e2e}。
结果分析： 整理三组数据，结合统计学方法得出结论。
